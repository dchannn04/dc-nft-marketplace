# dc-nft-marketplace
A decentralized platform to buy, list and sell NFTs. Built using Solidity, web3.js, React, Truffle, Hardhat, Metamask, and OpenZeppelin Contracts Library.


Dependencies Used:
Node.js 16.8.0, 
Hardhat v2.8.4,
React Router Dom,
IPFS HTTP Client, 
OpenZeppelin ERC721 Premade Contract
